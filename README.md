# StockEngine

A simple and clean stock market tracking app that helps you keep an eye on your favorite companies and see how they're doing.

## What Does This App Do?

StockEngine lets you:

- **See All Companies at a Glance** - View a list of popular companies with their stock prices and daily changes
- **Check Individual Stocks** - Click on any company to see detailed charts and price history
- **View AI Predictions** - Get future price predictions powered by smart algorithms
- **Track Market Trends** - See which stocks are going up or down, and spot market patterns
- **Manage Your Portfolio** - Keep track of stocks you're interested in (coming soon)

## Pages in the App

| Page | What It Shows |
|------|---------------|
| **Home (Market Overview)** | List of all companies with their current prices and changes |
| **Stock Details** | Detailed view of a single stock with charts and predictions |
| **Predictions** | AI-generated price forecasts for stocks |
| **Analytics** | Market trends, top gainers/losers, and trading signals |
| **Portfolio** | Your personal watchlist (coming soon) |
| **Settings** | App preferences |

## How to Use

1. **Sign Up** - Create an account with your email
2. **Log In** - Enter your credentials to access the app
3. **Browse Stocks** - Explore the market overview to see all companies
4. **Click on a Stock** - View detailed information and predictions
5. **Check Analytics** - See which stocks are performing best

## Tech Stack (For Developers)

This app is built with:

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **Charts**: Recharts
- **UI Components**: shadcn/ui

## Running Locally

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5000`
